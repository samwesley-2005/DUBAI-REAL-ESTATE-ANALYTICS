 Dubai Real Estate Analytics 

 Project Overview
This project analyzes Dubai real estate data to identify pricing patterns, market distribution, and property valuation trends.
The objective was to extract meaningful business insights from structured property listings.

Objectives
- Analyze property price distribution
- Compare pricing across locations
- Evaluate property type composition
- Identify valuation drivers
- Study relationships between variables

  Analysis Performed
  Price Analysis  
  Location-Based Insights  
  Property Type Distribution  
  Correlation Analysis  
  Trend Evaluation

  Key Visualizations
- Average Price by Location
- Average Price by Property Type
- Property Type Distribution
- Size vs Price
- Price Distribution

  Key Insights
 Property prices vary significantly by location  
 Villas exhibit premium valuation trends  
 Apartments dominate market listings  
 Bedrooms and size influence price structure  
 Market pricing shows wide distribution range
